# pyRCX 

A pure python IRCX (https://www.ietf.org/archive/id/draft-pfenning-irc-extensions-04.txt) server originally written with Python 2.3

## Usage

Run the main file to start the server

`python start.py`

### Tests

`python -m unittest`

### Dependencies

None yet
